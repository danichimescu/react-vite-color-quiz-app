

## GitHub Repository Description

A small React + Vite homework project with interactive UI demos and a local JSON-backed quiz system.

### Key Features
- React 19 app bootstrapped with Vite
- Client-side routing via `react-router`
- Four main feature pages:
  - `ColorPicker` — select and preview a color
  - `Gradient` — build a gradient from two colors
  - `SimpleForm` — form with validation and submit handling
  - `Quiz` — create quiz questions and answer them
- Local JSON server support for quiz data via db.json and `json-server-auth`

### Stack
- React
- Vite
- React Router
- JSON Server
- ESLint

### Useful Scripts
- `npm run dev` — start Vite dev server
- `npm run build` — build production bundle
- `npm run preview` — preview production build
- `npm run lint` — run ESLint
- `npm run server` — run JSON server backend
